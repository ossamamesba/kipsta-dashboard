# 🏆 KIPSTA Linear Management Dashboard

## 📌 Description
Ce projet est une interface web permettant de gérer et suivre le pointage linéaire de l’univers KIPSTA chez Decathlon.

L’objectif est d’améliorer la performance du linéaire et d’atteindre les 20 points gagnants KIPSTA.

---

## 🎯 Objectifs
- Suivre la disponibilité des produits
- Vérifier la conformité du facing
- Contrôler les prix et étiquettes
- Respecter le planogramme

---

## 💻 Fonctionnalités
- Pointage hebdomadaire par univers
- Suivi des scores par collaborateur
- Visualisation des performances (graphiques)
- Identification des axes d’amélioration
- Export des données en CSV

---

## 🏬 Univers concernés
- Foot Adulte
- Foot Junior
- Ballon Foot
- Basketball
- Tennis
- Padel
- Golf
- Volleyball
